  MCD — Modèle Conceptuel des Données — XFILES

ENTITÉS
-------

  FILIERE
    - code_filiere  (identifiant)
    - nom_filiere

  MODULE
    - id_module     (identifiant)
    - nom_module
    - code_module
    - semestre

  UTILISATEUR
    - id_user       (identifiant)
    - nom
    - login         (identifiant secondaire, unique)
    - mot_de_passe_hash
    - role          (etudiant | admin)
    - avatar
    - date_inscription

  DOCUMENT
    - id_document   (identifiant)
    - titre
    - nom_fichier
    - donnees_fichier (BLOB)
    - type_mime
    - taille_fichier
    - type_doc      (cours | td | tp | examen | resume)
    - statut        (en_attente | approuve | rejete)
    - motif_rejet
    - date_ajout

  COMMENTAIRE
    - id_commentaire (identifiant)
    - contenu
    - note          (1..5)
    - date_creation


ASSOCIATIONS
------------

  APPARTENIR (1,1) FILIERE —— (1,n) MODULE
    Une filière possède plusieurs modules
    Un module appartient à une seule filière

  INSCRIRE (0,1) FILIERE —— (0,n) UTILISATEUR
    Un utilisateur peut s'inscrire dans une filière (optionnel)
    Une filière peut avoir plusieurs utilisateurs

  UPLOADER (1,1) UTILISATEUR —— (0,n) DOCUMENT
    Un utilisateur peut uploader plusieurs documents
    Chaque document est uploadé par un seul utilisateur

  RATTACHER (1,1) MODULE —— (0,n) DOCUMENT
    Un module peut avoir plusieurs documents
    Chaque document est rattaché à un seul module

  COMMENTER (1,1) UTILISATEUR —— (0,n) COMMENTAIRE
    Un utilisateur peut écrire plusieurs commentaires
    Chaque commentaire est écrit par un seul utilisateur

  CONCERNER (1,1) DOCUMENT —— (0,n) COMMENTAIRE
    Un document peut avoir plusieurs commentaires
    Chaque commentaire concerne un seul document


CARDINALITÉS RÉSUMÉES
---------------------

  FILIERE  ——1,n—— APPARTENIR ——1,1—— MODULE
  FILIERE  ——0,n—— INSCRIRE   ——0,1—— UTILISATEUR
  UTILISATEUR ——0,n—— UPLOADER  ——1,1—— DOCUMENT
  MODULE   ——0,n—— RATTACHER  ——1,1—— DOCUMENT
  UTILISATEUR ——0,n—— COMMENTER ——1,1—— COMMENTAIRE
  DOCUMENT ——0,n—— CONCERNER  ——1,1—— COMMENTAIRE
